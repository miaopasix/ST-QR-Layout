# QR Layout Customizer

SillyTavern 快捷回复栏布局自定义扩展。

## 功能

- **布局模式**：支持弹性换行 / 按行数排列
- **按钮缩放**：整体缩放（50%-150%）
- **边距调节**：上下 / 左右间距可调
- **文字大小**：全局字体大小 + 每个按钮独立字号
- **按钮排序**：长按拖拽调整按钮位置，支持自动滚动和滚轮滚动
- **折叠按钮**：将按钮折叠到弹窗，弹窗从右往左、从下往上排列
- **折叠按钮自定义**：支持自定义图标（7种可选）、颜色、大小，间距可手动输入
- **折叠弹窗主题同步**：折叠弹窗背景、透明度、模糊度与 QR 栏/文字输入栏实时同步
- **空折叠提示**：折叠弹窗无按钮时点击弹出"这里没按钮啦~"提示
- **预设系统**：保存 / 加载 / 删除布局预设（预设完全隔离，互不干扰）
- **颜色与加粗**：每个按钮可独立设置颜色、加粗、字号、位置偏移
- **面板主题**：跟随酒馆 / 浅色 / 深色 三种主题，跟随酒馆模式完全同步 ST 的背景色、文字色、边框色、模糊度、输入框样式、滑块样式、复选框样式
- **开启动画**：设置面板、二级菜单展开均有平滑动画
- **手机兼容**：设置面板支持触摸拖拽移动
- **酒馆助手兼容**：支持收纳酒馆助手脚本创建的独立按钮（如小铅笔遥控器）

## 安装

### 方式一：从 URL 安装
1. 在 SilleyTavern 的扩展管理中，选择"从 URL 安装"
2. 输入本仓库地址：`https://github.com/miaopasix/ST-QR-Layout`
3. 安装完成后，重启 SillyTavern

### 方式二：手动安装
1. 从 [Releases](https://github.com/miaopasix/ST-QR-Layout/releases) 下载最新版 zip
2. 解压到 `SillyTavern/public/scripts/extensions/third-party/` 目录下
3. 重启 SillyTavern

## 使用

1. 打开设置面板 -> QR 布局 -> 点击「打开设置面板」
2. 勾选「启用布局自定义」后，所有设置项生效
3. 取消勾选将恢复默认样式
4. 在「面板主题」中选择跟随酒馆 / 浅色 / 深色
